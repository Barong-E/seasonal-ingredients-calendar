제철음식 알리미 프로젝트에 유저 행동 분석을 위해 구글 애널리틱스(GA4)와 마이크로소프트 클레어리티(MS Clarity)를 연동하려고 해.
다음 조건에 맞춰서 코드를 작성하고 적용해 줘:
스크립트 삽입: 프로젝트의 모든 페이지에서 작동할 수 있도록 공통 최상위 파일(예: Next.js의 경우 app/layout.tsx 또는 _document.js, 일반 HTML인 경우 index.html의 <head> 태그 안)에 두 가지 추적 스크립트를 추가해 줘.
보안 및 환경 변수: GA4 측정 ID와 Clarity 프로젝트 ID는 코드에 직접 적지 말고, .env 파일 같은 환경 변수를 사용해서 관리할 수 있도록 세팅해 줘.
성능 최적화: 스크립트가 로딩되느라 웹사이트가 느려지면 안 되니까, 비동기(async/defer) 방식이나 프레임워크에서 권장하는 최적화된 스크립트 로딩 방식을 사용해 줘.
안내: 코드를 다 작성한 후, 내가 실제 발급받은 GA4 ID와 Clarity ID를 어느 파일, 어떤 변수명에 붙여넣으면 되는지 친절하게 순서대로 알려줘.
