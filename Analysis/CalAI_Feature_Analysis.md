# 📱 Cal AI 앱 — 전체 기능 상세 분석 보고서

> Cal AI는 음식 사진을 촬영하면 AI가 자동으로 칼로리와 영양 성분을 분석해주는 AI 기반 식단 관리 앱입니다.

---

## 1. 🚀 온보딩 (Onboarding Flow)

사용자가 앱을 처음 실행하면 **퀴즈 스타일의 개인화 설정 과정**을 거칩니다.

| 수집 항목 | 설명 |
|:---|:---|
| **기본 정보** | 성별, 나이, 키, 현재 체중 |
| **목표 설정** | 감량 / 유지 / 증량 중 선택 |
| **목표 체중** | 구체적인 숫자 입력 |
| **감량 속도** | 느리게 / 보통 / 빠르게 → 슬라이더로 조정하면 예상 달성 기간이 실시간 변화 |
| **활동 수준** | 거의 없음 / 가벼운 운동 / 보통 / 활발 / 매우 활발 |
| **식단 선호** | 일반식 / 키토 / 비건 / 저탄수화물 등 |
| **식사 횟수** | 하루 몇 끼를 먹는지 |

### 온보딩의 핵심 전략
- **"첫 성공 경험"** 을 빠르게 제공: 온보딩 직후 바로 첫 번째 식사를 사진으로 기록하도록 유도
- 상세한 정보 수집을 통해 **개인화된 일일 칼로리/영양소 목표**를 자동 계산
- 유료 결제 화면은 온보딩이 **모두 끝난 후**에 나타남 (사용자가 이미 투자한 시간 때문에 결제 전환율 상승)

## 2. 📸 AI 사진 기반 식사 분석 (핵심 기능)

Cal AI의 **가장 핵심적인 차별화 기능**입니다.

### 작동 방식
```
사진 촬영 → AI가 음식 종류 인식 → 양(부피) 추정 → 칼로리 & 영양소 계산 → 결과 표시
```

### 분석 결과로 제공되는 정보
| 항목 | 설명 |
|:---|:---|
| **칼로리 (kcal)** | 총 열량 |
| **단백질 (g)** | Protein |
| **탄수화물 (g)** | Carbs |
| **지방 (g)** | Fat |

### AI 보정 루프 (Correction Loop)
- AI가 분석한 결과가 틀렸을 경우, 사용자가 **수동으로 수정** 가능
- 재료 변경, 양 조절, 음식 종류 재선택 가능
- 사용자의 수정 데이터가 **AI 모델 학습에 반영**되어 시간이 갈수록 정확도 향상

### 정확도 현황
| 음식 유형 | 정확도 |
|:---|:---|
| 단일 식재료 (사과, 바나나 등) | ⭐⭐⭐⭐ 높음 |
| 간단한 요리 (샐러드, 구운 닭가슴살) | ⭐⭐⭐ 보통~높음 |
| 복잡한 혼합 요리 (비빔밥, 카레 등) | ⭐⭐ 낮음 (~25~35% 오차) |
| 소스가 많은 요리 | ⭐ 매우 낮음 |

## 3. 🔤 다양한 식사 입력 방식

사진 외에도 여러 방법으로 식사를 기록할 수 있습니다.

| 입력 방식 | 설명 |
|:---|:---|
| **📷 AI 사진 스캔** | 음식 사진을 찍으면 자동 분석 |
| **📊 바코드 스캔** | 포장 식품의 바코드를 스캔하면 영양 정보 자동 입력 |
| **🏷️ 영양 성분표 인식** | 영양 성분 라벨을 카메라로 촬영하면 자동 인식 |
| **🎤 음성 입력** | "터키 샌드위치와 사과를 먹었어"라고 말하면 기록 |
| **🔍 검색 입력** | 100만 개 이상의 음식 데이터베이스에서 직접 검색 |
| **✏️ 수동 입력** | 커스텀 음식의 영양소를 직접 입력 |

## 4. 📊 대시보드 & 일일 요약

### 메인 대시보드 구성
- **원형 차트/링 차트**: 일일 칼로리 목표 대비 현재 섭취량을 시각적으로 표시
- **매크로 바**: 단백질 / 탄수화물 / 지방 각각의 목표 대비 섭취량
- **식사별 분류**: 아침 / 점심 / 저녁 / 간식으로 구분하여 기록
- **잔여 칼로리**: "오늘 아직 ○○○ kcal 더 먹을 수 있어요" 표시

### 부가 영양 정보 (스와이프하면 보이는 항목)
- 식이섬유 (Fiber)
- 나트륨 (Sodium)
- 당류 (Sugar)

> [!NOTE]
> Cal AI는 매크로(탄단지) 중심 앱이므로, 비타민·미네랄 등 **미량영양소 추적은 지원하지 않습니다**.
> 미량영양소가 필요하면 Cronometer나 Nutrola 같은 앱을 추천합니다.

## 15. ⚠️ 알려진 한계점 & 비판

| 한계점 | 상세 설명 |
|:---|:---|
| **복잡한 음식 정확도 낮음** | 비빔밥, 찌개 등 한국 음식은 인식이 어려움 (25~35% 오차) |
| **미량영양소 미지원** | 비타민, 미네랄 등 세부 영양소 추적 불가 |
| **수정 UI 불편** | AI가 틀렸을 때 수동 수정하는 과정이 번거로움 |
| **결제 투명성 부족** | 가격이 온보딩 후에만 표시되고, 사용자마다 다름 |
| **구독 해지 어려움** | 구독 취소 및 환불 프로세스에 대한 불만 다수 |
| **앱 가용성 변동** | 2026년 기준 앱의 가용성이 변동 중이라는 보고 있음 |

## 🎯 제철 식재료 캘린더 프로젝트에 적용 가능한 기능 제안

현재 프로젝트의 특성을 고려했을 때, Cal AI에서 차용할 수 있는 기능들입니다:

| 기능 | 적용 아이디어 | 난이도 |
|:---|:---|:---|
| **📸 AI 사진 → 식재료 인식** | 사진을 찍으면 "이건 OO 식재료입니다" + 제철 여부 + 칼로리/영양소 표시 | 🔴 높음 (API 연동 필요) |
| **📊 영양소 정보 표시** | 각 제철 식재료 상세 페이지에 칼로리/영양소 정보 추가 | 🟢 낮음 (정적 데이터) |
| **🔍 식재료 검색 + 영양소 필터** | "단백질이 높은 제철 식재료" 같은 필터 검색 | 🟡 보통 |
| **📈 식단 기록 & 요약** | 오늘 먹은 제철 식재료 기반 간단한 영양 요약 | 🟡 보통 |
| **🏆 제철 식재료 도전** | "이번 달 제철 식재료 10가지 먹어보기" 같은 게이미피케이션 | 🟢 낮음 |

> 어떤 기능을 프로젝트에 넣고 싶으신지 구체적으로 알려주시면, 해당 기능에 맞는 작업 계획서를 만들어드릴게요!
