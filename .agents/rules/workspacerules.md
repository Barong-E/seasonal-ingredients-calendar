## 프로젝트 맞춤 설정 및 자동화 (Customizations & Automation)
- **[중요] 빌드 폴더 수정 절대 금지**: `docs/` 폴더는 Vite 빌드 결과물이 저장되는 폴더임. 이 폴더 안의 파일을 수정하면 빌드 시 초기화되거나 안드로이드 앱과 동기화되지 않으므로 절대 직접 수정하지 말 것.
- **수정 위치 준수**: 반드시 루트 디렉토리에 있는 원본 파일(`index.html`, `setting.html` 등)이나 `assets/` 폴더 내의 파일만 수정할 것.
- **작업 후 자동 반영 (필수)**: 코드를 수정한 후에는 별도 안내 없이 에이전트가 직접 터미널 도구를 사용해 다음 두 가지 작업을 **반드시 순서대로 모두 실행**할 것. 하나라도 누락되어서는 안 됨.
  1) **안드로이드 동기화**: `npm run build && npx cap sync`
  2) **Git 원격지 푸시**: `git add . && git commit -m "[자동] 작업 내용 요약" && git push`
